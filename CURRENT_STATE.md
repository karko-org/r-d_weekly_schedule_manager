## ⚠️ 알려진 제약사항

### 브라우저 호환성
- ✅ Chrome (v86+), Edge (v86+), Opera (v72+)
- ❌ Firefox, Safari — File System Access API 미지원 → UI가 자동으로 기능 비활성화 + 경고 표시

### 기능적 제약
- **업무는 한 달을 넘길 수 없음** — 설계상 의도 (각 업무는 단일 월 내에만 존재)
- **모바일 미지원** — 데스크탑 전용 (1400px 펼치기 모드 고려 시 사실상 필수)
- **오프라인 전용 LocalStorage** — 다른 기기와 동기화 없음 (엑셀 파일이 sync source)
- **Task ID는 import 시 재생성** — 같은 파일 재import 시 notes는 셀 코멘트에서 복원되지만 id는 새로 부여됨

### 알려진 엣지 케이스
- 엑셀에 같은 월이 여러 번 있는 경우 파싱 처리 모호 (발생 가능성 낮음)
- 편집 중인 tw-item에서 주 전환(`이번주 ↔ 다음주`)하면 편집 모드 초기화
- 브리핑 모드에서 월 이동 후 편집 뷰로 돌아가면 편집 뷰 월은 변경되지 않음 (의도)
